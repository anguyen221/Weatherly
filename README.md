# Weatherly

**Weatherly** is a full-featured Flutter weather application powered by Firebase and OpenWeatherMap. It offers real-time weather data, forecasts, map visualizations, alerts, user customization, and a social component to report and share weather conditions.

# Features

- **Login/Signup Screen**: Users can register and log in securely to access the app.
- **Home Screen**: Displays real-time weather based on the user’s saved location.
- **Forecast Screen**: Shows hourly and weekly weather forecasts.
- **Map Screen**: Interactive map showing layers for precipitation, wind, clouds, and temperature.
- **Alerts Screen**: Allows users to opt-in for push notifications on severe weather.
- **Themes Customizer Screen**: Users can apply mood-based UI themes.
- **Settings Screen**: Users can update their username.
- **Community Reports Screen**: Submit and view local weather reports from other users.
- **Share Weather Screen**: Share real-time weather conditions to social media platforms.
